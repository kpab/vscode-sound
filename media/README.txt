# メディアファイルについて

このフォルダには以下のファイルを配置してください：

## アイコン
- icon.png - 拡張機能のアイコン（128x128ピクセル推奨、PNG形式）

## サウンドファイル（sounds/ フォルダ内）
- key-press.mp3 - 標準的なキー押下音
- enter.mp3 - Enterキー用
- tab.mp3 - Tabキー用
- space.mp3 - スペースキー用
- backspace.mp3 - Backspaceキー用

これらの音源ファイルは以下のサイトから入手できます：
- 効果音ラボ: https://soundeffect-lab.info/
- DOVA-SYNDROME: https://dova-s.jp/
- Freesound: https://freesound.org/
- SoundBible: https://soundbible.com/
- Zapsplat: https://www.zapsplat.com/
